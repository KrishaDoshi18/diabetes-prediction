# 🩺 Diabetes Prediction using Machine Learning

## 📌 Overview

This project predicts whether a person is diabetic or not using machine learning algorithms.

## 🚀 Features

* Data preprocessing and cleaning
* Multiple ML models:

  * Logistic Regression
  * SVM
  * KNN
  * Decision Tree
  * Random Forest
  * AdaBoost
  * Naïve Bayes
* Model comparison and evaluation

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 📊 Results

* Compared multiple models based on accuracy
* Selected best-performing model

## 📂 Project Structure

```
├── diabetes.csv
├── models.ipynb
├── README.md
```

## ▶️ How to Run

1. Clone repo
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run:

   ```
   python models.ipynb
   ```
